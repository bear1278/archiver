# Archiver

Архиватор файлов на RLE с консольным интерфейсом Buble Tea.

[![Go Version](https://img.shields.io/badge/Go-1.21+-00ADD8?style=flat&logo=go)](https://golang.org)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 🔧 Установка

### Из исходного кода

```bash
# Клонирование репозитория
git clone https://github.com/bear1278/archiver.git
cd archiver

# Сборка проекта
go build .

# Запуск
./archiver
```

# 📖 Использование

### Меню

```bash
===== Архиватор =====

> Сжать файл
  Распаковать файл
  Выход
Используйте стрелки для навигации и enter для выбора
Нажмите q для выхода

```

### Сжатие файла
```bash
===== Архиватор =====

Введите к файлу для сжатия:
test.txt_
Enter для подтверждения, Esc для возврата в меню
```


```bash
===== Архиватор =====

Введите к файлу для распаковки:
test.txt_
Enter для подтверждения, Esc для возврата в меню
```
